=================
下载安装
=================
 
 
使用前提
-------------

-  安装Python

-  安装pandas

-  lxml也是必须的，正常情况下安装了Anaconda后无须单独安装，如果没有可执行：pip install lxml


下载安装	
-------------

.. code:: python

        pip install mineshare

 
版本升级 
------------- 

.. code:: python

        pip install mineshare --upgrade
