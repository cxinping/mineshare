MineShare¶

Note

MineShare是是一个免费、开源的python虚拟币数据接口包。主要实现对虚拟币等金融数据从 数据采集 、 清洗加工 到 数据存储 的过程，提供便于分析的数据。目前MinShare是在Python3.*基础上开发的，没有对Python2.*进行兼容测试，如果有问题，请联系作者，欢迎交流与指教。

MineShare从开发到发布，得到了Tushare(http://tushare.waditu.com/)作者Jimmy的大力支持，再次表示感谢，开源软件就是这样赠人玫瑰，手有余香。

MineShare的数据主要来源于网络，如果在使用过程碰到数据无法获取或发生数据错误的情况请联系我，我的微信是: xinpingws ，请填好备注：mineshare学习。
